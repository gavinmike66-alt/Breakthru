# Note Parser Plugin

Parses free-text field sales notes into structured data.

## Skills

- **daily-report-parser** — Parses rep daily reports into account visit records with orders, placements, follow-ups, and competitive intel
- **ride-with-recap** — Parses ride-with notes into structured coaching recaps with strengths, development areas, and action items

## Commands

- `/note-parser:parse-notes [rep-name]` — Parse daily notes for a specific rep
- `/note-parser:recap [rep-name]` — Generate a ride-with recap from field notes
