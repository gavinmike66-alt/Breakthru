# Program Tracker Plugin

Tracks and consolidates 21 supplier programs across varied reporting formats into a unified view.

## Skills

- **program-consolidation** — Merges program data from multiple supplier formats into a standard schema with status tracking
- **program-status** — Checks program progress, flags at-risk programs and upcoming deadlines

## Commands

- `/program-tracker:programs [supplier-name]` — Show program status overview, optionally filtered by supplier
- `/program-tracker:consolidate` — Consolidate provided program data from multiple formats
