Consolidate the provided supplier program data into a unified tracking view.

Map varied supplier formats to the standard schema: supplier, program name, period, goal type, target, progress, % to goal, incentive, key accounts, rep assignment, status, notes.

Group by status (At Risk first). Flag programs below 70% time-proportional goal as **[AT RISK]** and those expiring within 2 weeks as **[DEADLINE]**.
