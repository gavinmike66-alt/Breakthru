# Rep Performance Analysis

Analyze performance for "$ARGUMENTS".

## Output Format

### Activity Metrics
- Accounts visited vs target
- Call frequency and coverage
- New account activity
- Order volume and trends

### Program Execution
- Programs assigned and their status
- Contribution to team program goals
- Programs at risk under this rep

### Strengths
What this rep is doing well, supported by data

### Development Areas
Where performance gaps exist, with specific examples

### Recommended Actions
Concrete next steps for coaching or support — tie back to observed data

## Instructions

1. Be data-driven — every observation should reference specific metrics or examples
2. Frame development areas as opportunities, not criticisms
3. If comparing to team averages, note the comparison explicitly
4. Suggest specific coaching actions, not generic advice
