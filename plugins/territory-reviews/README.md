# Territory Reviews Plugin

Weekly territory performance reporting and individual rep analysis for Palm Beach County.

## Skills

- **weekly-review** — Builds a weekly territory review with snapshot metrics, rep summaries, program progress, and action items
- **rep-performance** — Analyzes individual rep performance across activity, sales, and program execution

## Commands

- `/territory-reviews:weekly` — Generate a weekly territory review
- `/territory-reviews:rep-review [rep-name]` — Generate a performance review for a specific rep
